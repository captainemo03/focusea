Contact

Contact Focusea for product, privacy, advertising, legal notice and security questions.

For now, Focusea uses the public project issue tracker as its contact channel so feedback and fixes can be tracked clearly.

Open Focusea GitHub Issues

What to include

For product feedback, include the page name, what you expected and what happened. For privacy or security reports, avoid posting secrets publicly; describe the issue safely and request private follow-up if needed.

Policy requests

For privacy, advertising, content removal, correction or legal notice requests, include the relevant page URL, the exact concern and a safe contact method for follow-up.

Security reporting

Focusea also publishes a security contact file at security.txt.
